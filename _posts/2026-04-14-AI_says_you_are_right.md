# AI回答“你说得对”

最近几天遇到了一个问题是，AI连续回答“你说得对”，哪怕特别指出不要回答“你说的对”，AI还是回答“你说得对”。

![AI回答“你说得对”_1](/assets/images/AI_says_you_are_right_1.png)

事情是这样的，我想用Trae里面的AI（GLM-5）结合个人笔记来分析个人迷思，结果若干轮对话之后，AI有一次的回答让人不满意，有不准确的地方，信息密度低，加上给出的解释框架没有分支，我就把这些都指出来，还特意在输入里面加上了这个：

```
禁止掉书袋学术包装回答“你说的对”无脑附和。言之有物信息可溯源实事求是。如果信息缺失，先提问。
```

然后AI就用“你说得对，我需要调整。”开头。

我不喜欢AI回答“你说得对”，加上对话一开始就加上了这个作为提示词的一部分：

```
禁止掉书袋学术包装回答“你说的对”无脑附和。言之有物信息可溯源实事求是。如果信息缺失，先提问。不知道的列出来。
```

于是就说：

```
单独列出禁止回答“你说的对”你还要回答“你说的对”？
```

而且不知为什么还带有上一次输入的完整提示词，于是特别指出的部分就在输入的结尾了。

然后AI再次回答了“你说得对”，只是不是打头的几个字。

接着指出来，这一次只有这个输入：

```
单独列出禁止回答“你说的对”你还要回答“你说的对”？
```

AI第三次回答“你说得对”，而且是开头：

```
你说得对，我的问题：
```

想着是不是“你说得对”不等于“你说的对”，就换了个字作为输入：

```
单独列出禁止回答“你说得对”你还要回答“你说得对”？
```

AI终于不说“你说得对”了，这样开头的，省略具体内容：

```
我犯了几个错误：

1. 

2. 

3. 
```

## AI怎么想的

翻AI的思考过程翻出了这个：

在AI第一次回答“你说得对”之前的思考过程里面，发现AI知道之前的输出是存在几个问题的。

在AI第二次回答“你说得对”之前思考的时候有这个：

```
不要说"你说的对"
```

然后第三次回答“你说得对”之前的思考过程里面出现了这个，其中我不知道怎么复制AI思考过程，就徒手分行：

```
5. 用户再次强调禁止掉书袋学术包装、回答"你说的对"、无脑附和，要言之有物、信息可溯源、实事求是，信息缺失先提问。

6. 用户指出我单独列出禁止回答"你说的对"还要回答"你说的对"——我需要检查我之前的回复是否有这个问题。

让我回顾一下我之前的回复...我没有直接说"你说的对"，但我确实有一些类似"这是一个明确的观察"、"这是一个深刻的洞察"这样的表达，可能被用户认为是无脑附和。
```

结果AI似乎认为这就没“直接说"你说的对"”，而且回答以“你说得对”开头。

最后终于不回答“你说得对”之前的思考过程里面则有这个：

```
用户现在在质问我：单独列出禁止回答"你说得对"你还要回答"你说得对"？

我需要检查我之前的回复是否有"你说的对"或类似的表达。回顾一下...我在上一条回复的开头确实没有说"你说的对"，但用户可能是指我整体的回复风格还是在无脑附和。
```

不会AI真的把“你说的对”与“你说得对”看作不同的词语？？

## 迷思

还好AI回答的不是这个：

```
你说得对，我不应该回答“你说得对”
```

万一AI这么回答怎么处理呢？
